# LINE通知の設定手順（保護者向け・所要 約20分）

学習結果（今日の問題数・模試の偏差値・連続日数）を保護者のLINEに届ける設定です。
**すべて無料**でできます（LINE Messaging APIの無料枠：月200通で十分足ります）。

> 参考：以前あった「LINE Notify」は2025年3月に終了したため、この手順ではLINE公式アカウント（Messaging API）を使います。

## 全体の流れ

```
学習アプリ → Google Apps Script（無料） → LINE公式アカウント → 保護者のLINE
```

## 手順1：LINE公式アカウント（Messaging API）を作る

1. https://developers.line.biz/console/ にLINEアカウントでログイン
2. 「プロバイダー作成」→ 名前は「家族」など何でもOK
3. 「チャネル作成」→「Messaging API」を選択
   - チャネル名：「学習アカデミー通知」など
   - 業種：個人 でOK
4. 作成後、**LINE公式アカウントマネージャー**（https://manager.line.biz/）側で作成される公式アカウントを、**保護者のLINEで友だち追加**する（チャネル設定画面にQRコードがあります）
5. チャネルの「Messaging API設定」タブで
   - **チャネルアクセストークン（長期）** を発行してコピー → 後で使います
6. 「あいさつメッセージ」「応答メッセージ」はオフ推奨（LINE公式アカウントマネージャーの応答設定）

## 手順2：自分のuserIdを調べる

送信先（保護者）の userId が必要です。いちばん簡単な方法：

1. LINE Developersコンソールのチャネル「基本設定」タブの一番下にある
   **「あなたのユーザーID」**（`U` から始まる文字列）をコピー
   ※これはチャネルを作った本人のuserIdです。通知を受け取る人がチャネル作成者本人ならこれでOK。

（家族グループに送りたい場合は、Botをグループに招待してWebhookでgroupIdを調べる方法もありますが、まずは個人宛てで始めるのがおすすめです）

## 手順3：Google Apps Script を作る

1. https://script.google.com/ → 「新しいプロジェクト」
2. エディタに、同じフォルダにある **`LINE通知_GAS.gs`** の中身を全部貼り付けて保存
3. 左メニュー「プロジェクトの設定」→ 一番下の「スクリプト プロパティ」で2つ追加：
   | プロパティ名 | 値 |
   |---|---|
   | `CHANNEL_ACCESS_TOKEN` | 手順1-5でコピーしたトークン |
   | `TO_ID` | 手順2でコピーした `U…` のID |
4. 「プロジェクトの設定」でタイムゾーンが **東京** になっているか確認（`appsscript.json` の timeZone が `Asia/Tokyo`）
5. エディタで関数 `setup` を選んで「実行」（初回は権限の承認画面が出ます→許可）
   - これで「毎晩21時のまとめ」「19時の未学習リマインド」のトリガーが作られ、LINEに準備完了メッセージが届きます

## 手順4：Webアプリとして公開（デプロイ）

1. 右上「デプロイ」→「新しいデプロイ」→ 種類「**ウェブアプリ**」
2. 設定：
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**（匿名を含む）
3. デプロイして表示される **ウェブアプリのURL**（`https://script.google.com/macros/s/…/exec`）をコピー

## 手順5：アプリに貼り付け

1. 学習アプリを開く → ⚙️設定 → 「💬 LINE通知」
2. 手順4のURLを貼り付けて「保存」
3. 「📨 テスト送信」→ LINEに「✅ 接続テストに成功しました！」が届けば完了 🎉

## 届く通知

| タイミング | 内容 |
|---|---|
| 模試が終わったとき | 得点・偏差値・判定・連続日数（即時） |
| 今日の目標を達成したとき | 達成メッセージ（即時） |
| 毎晩21時 | 今日のまとめ（問題数・正答率・模試結果・連続日数） |
| 19時 | その日まだ学習していなければリマインド |

## うまくいかないとき

- テスト送信してもLINEに届かない
  - スクリプトプロパティの `CHANNEL_ACCESS_TOKEN` / `TO_ID` を再確認
  - 公式アカウントを**友だち追加**したか確認（ブロックしていないか）
  - GASの「実行数」画面（時計アイコン）でdoPostのエラーを確認
- コードを修正したら、**「デプロイ」→「デプロイを管理」→ 編集 → 新バージョン**で更新（URLは変わりません）
